# devin-training-sandbox ## Setup npm install npm test # verify all tests pass before making changes ## Tech Stack - Runtime: Node.js 20 LTS - Framework: Express.js - Testing: Jest - Language: JavaScript (ES6+) ## Conventions - Route files: src/routes/[name].js (each exports an Express Router) - Test files: **tests**/[name].test.js - Formatting: Prettier (run npx prettier --write . before committing) - Branching: feature/[short-description] - Commit messages: Use conventional commit format (feat:, fix:, test:, docs:) ## Off-Limits (Do NOT modify these files) - **tests**/existing.test.js (baseline test suite) - package.json scripts section - src/middleware/auth.js (authentication middleware) - .github/ directory (CI/CD workflows) ## Reference Patterns - For new routes, follow the pattern in src/routes/users.js - For new tests, follow the pattern in **tests**/users.test.js - All routes must be registered in src/app.js ## Known Quirks - Auth middleware must be mounted before route handlers in app.js - Tests run in parallel by default; use --runInBand if order matters - The .env.example file has all required environment variable names
